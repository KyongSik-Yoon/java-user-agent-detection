        public Locale(java.lang.String l,
              java.lang.String c)
        This constructor defines both kanguage and country from their string representation. Their string representation is the 2-letter ISO code for the country and the language. Thay can be provided in either uppercase or lowercase form but not in a mixed-case form. If one or both parameters cannot be found in the known languages/countries, they are set to their respective UNKNOWN value.
